# Day 01 – DevOps & Cloud Learning Plan

## 1. My Current Level

I am a fresher with hands-on exposure to Linux, Git, and basic cloud concepts. I have started working on real DevOps tasks but want to become job-ready with strong practical skills.

---

## 2. My Understanding of DevOps & Cloud

DevOps is a culture and set of practices that focuses on automating and improving the process of building, testing, and deploying applications. It emphasizes collaboration between development and operations teams.

Cloud Engineering involves using platforms like AWS to provision, manage, and scale infrastructure on demand instead of relying on physical servers.

---

## 3. Why I Am Learning DevOps & Cloud

- To become job-ready in a high-demand field
- To build real-world infrastructure and deployment skills
- To transition from learning basics to working on production-level systems

---

## 4. My 90-Day Goals

1. Deploy a multi-tier application using Docker and Kubernetes
2. Build and automate CI/CD pipelines using GitHub Actions or Jenkins
3. Gain strong troubleshooting skills in Linux, networking, and cloud environments
